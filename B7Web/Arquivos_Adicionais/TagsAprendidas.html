Todas as Tags aprendidas

html:5

Primeira tag que se usa ao criar a pagina, ao usala, automaticamente se cria todas as tags padrões e necessarias para uma pagina de html como o body, head, title e etc.

br

"br" serve para pular linhas, a quantidade de "br" que usar vai ser a quantidade de linhas que pula

h1 h2...h6

Os "h" são usados para criar textos com o tamanho definido de acordo com o "h" que se usou, quanto maior numero do "h" menor a letra

P

O P cria um novo paragrado no texto, pulando de linha automaticamente

a href="link"

O "a" ele ancora um link dentro dele

img src="" alt="link da imagem"

O "img" serve para por o link ou endereço de onde está a imagem para ela aparecer na pagina

button

Cria um botão

"ul" "ol" seguido de "li"

"ul" cria uma lista sem ordenação na frente dos itens
"ol" criar uma listra ordenada
"li" é cada item que vai dentro de uma das listras acima, ficando por exemplo:
ol
li>item 1/li
li>item 2/li
/ol>
Na pratica ficaria assim
  1. item 1
  2. item 2
